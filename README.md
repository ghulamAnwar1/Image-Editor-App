# Image Editor App

A web-based image editor with filters, presets, and download functionality built using HTML, CSS, and JavaScript.

## Features
- Upload images
- Apply multiple filters like brightness, contrast, saturation, blur, etc.
- Presets for quick edits
- Reset filters to default
- Download edited image

## How to Use
1. Open `index.html` in a browser.
2. Choose an image to edit.
3. Adjust filters using sliders or apply presets.
4. Download the edited image.

## Technologies
- HTML
- CSS
- JavaScript
